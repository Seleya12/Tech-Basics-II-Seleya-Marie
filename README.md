# Two-Player Arduino Reaction Game by Seleya & Marie

## Project Description
This project is a two-player reaction and memory game built with an Arduino Uno.  
Two players compete against each other by reacting to visual LED signals and repeating LED
sequences as quickly and accurately as possible.  
The game measures reaction time and visually indicates the winner.

## Hardware Requirements
- Arduino Uno
- 3 Breadboards
- 7 Buttons (3 per player + one to start the game)
- 6 LEDs (3 per player: green, yellow, red)
- 6 × 220 Ω resistors
- Jumper wires
- USB cable (power supply via PC)

## How to Run
1. Connect the hardware components according to the pin assignments described in the process
2. documentation.
3. Upload the Arduino code from `arduino/game_code.ino` to the Arduino Uno.
4. Power the Arduino via USB connection.
5. Use the start button to select a game level and begin playing.
